# Style Memory

## Canonical style name
- `Neo-Classic OS UI`

## Alternate labels
- `Retro System Chrome`
- `Mac OS 9 Revival`
- `Arcade Workstation UI`

## Style characteristics
- Skeuomorphic depth with beveled borders and inset highlights.
- Dark navy system-chrome palette with subtle gradients.
- Chunky, pixel-adjacent mono typography for UI controls.
- Strong pressed/raised states for tactile interaction.
